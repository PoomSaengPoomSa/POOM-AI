# 📈 Explainable KOR-rate: 한국은행 기준금리 방향 예측 및 XAI 심층 분석

## 💡 프로젝트 배경 및 목적
한국은행의 기준금리는 국가 경제의 향방을 결정하는 가장 강력한 통화정책 수단입니다. 금리 변동은 환율, 가계 부채, 기업 투자 심리 등에 즉각적인 영향을 미치기 때문에 그 방향성(인하, 동결, 인상)을 정확히 예측하는 것은 금융 의사결정의 핵심입니다.

본 프로젝트는 단순히 금리 방향을 맞추는 '예측 성능'에만 집중하지 않고, **"왜 모델이 특정 시점에 금리 인하 혹은 인상을 예측했는가?"**에 대한 근거를 제시하는 것에 초점을 맞췄습니다. 금융 도메인에서 인공지능이 실무자의 신뢰를 얻기 위해 필수적인 **설명 가능한 AI(XAI, Explainable AI)** 기술을 접목하여, 복잡한 거시 경제 지표 사이의 상관관계를 투명하게 분석합니다.

---

## 🚀 핵심 기술 및 방법론

### 1) 데이터 전처리 및 차원 축소 (VIF 기반)
거시 경제 지표(금리, 물가, 환율, 통화량 등)는 서로 밀접하게 연관되어 있어 **다중공선성(Multicollinearity)** 문제가 발생하기 쉽습니다. 이는 모델의 해석력을 떨어뜨리는 주요 원인이 됩니다. 본 시스템은 `dimensionality_reduction.py`를 통해 VIF 지수를 계산하고, 임계치를 넘는 변수를 반복적으로 제거하여 모델의 안정성을 확보했습니다.

### 2) 앙상블 기반 피처 셀렉션 (RF Voting)
단순한 상관관계 분석을 넘어, 20개의 서로 다른 시드를 가진 Random Forest 모델을 활용해 **피처 중요도 투표(Voting)**를 실시합니다. 이를 통해 특정 데이터 분포에 치우치지 않고, 금리 결정에 가장 지배적인 영향력을 행사하는 상위 핵심 변수(Top-K)만을 엄선하여 학습에 사용합니다.

### 3) 클래스 불균형(Class Imbalance) 대응
과거 금리 데이터는 '동결' 사례가 '인상'이나 '인하'에 비해 압도적으로 많습니다. 이러한 불균형을 해소하지 않으면 모델은 항상 '동결'로만 예측하려는 경향을 보입니다. 본 프로젝트는 학습 시 **클래스별 가중치(인하: 5.0, 동결: 1.0, 인상: 5.0)**를 부여하여 소수 클래스(인하/인상)에 대한 탐지력을 획기적으로 높였습니다.

### 4) SHAP 기반 모델 해석 시스템
학습된 블랙박스 모델의 내부를 들여다보기 위해 **SHAP(SHapley Additive exPlanations)** 방법론을 도입했습니다.
- **Global Interpretation**: 전체 기간 동안 금리 변동에 기여한 변수들의 기여도를 산출합니다.
- **Local Interpretation (Error Analysis)**: 모델이 실제 금리 방향과 다르게 예측한 '오분류 케이스'를 자동 추출하여, 해당 시점에 모델을 혼란스럽게 만든 경제 지표가 무엇인지 워터폴(Waterfall) 차트로 정밀 분석합니다.

## 📂 프로젝트 구조
```txt
Explainable_KOR-rate/
├── run.py                    # 전체 파이프라인(학습-테스트-해석) 통합 실행 스크립트
├── train.py                  # 데이터 전처리, VIF 차원축소, 피처 셀렉션 및 모델 학습
├── test.py                   # 테스트 데이터를 활용한 모델 성능 검증 및 지표 산출
├── explain.py                # SHAP을 활용한 모델 해석 및 시각화 이미지 생성
├── interpret_xai.py          # 생성된 SHAP 결과를 텍스트 리포트로 요약 및 해석
├── model.py                  # InterestRateEnsembleModel 클래스 정의 (XGBoost/CatBoost 기반)
├── dimensionality_reduction.py # VIF 알고리즘 및 차원 축소 로직 포함
├── utils/
│   ├── get_data.py           # 데이터 로드 및 기본적인 데이터 정제
│   └── preprocess.py         # 스케일링 및 타겟 라벨링 로직
├── data/
│   ├── final_dataset.csv     # 최종 학습용 데이터셋
│   └── metadata.csv          # 변수별 한글/영문 매핑 정보
├── models/                   # 학습된 모델(.pkl) 및 선택된 피처 리스트 저장
└── results/                  # 시각화 이미지(.png) 및 분석 리포트(.csv, .md) 저장 폴더
```

## 💻 설치 및 실행 방법
```bash
# 1. 환경 구축
# Python 3.11+ 환경 권장
pip install -r requirements.txt

# 2. 전체 파이프라인 실행
# 데이터 처리부터 모델 학습, SHAP 해석까지 모든 과정이 순차적으로 진행됩니다.
python run.py

# 3. 주요 모듈 개별 실행 (선택 사항)
# 모델 학습만 진행할 경우
python train.py

# 학습된 모델을 바탕으로 XAI 분석만 수행할 경우
python explain.py
```

## 📊 주요 분석 결과 (XAI Output)

프로젝트 실행 시 `results/` 폴더에 생성되는 결과물은 다음과 같은 의미를 갖습니다.

1. **Beeswarm Plot**: 각 경제 지표의 수치가 높고 낮음이 금리 인하/인상 확률을 어느 방향으로 밀어냈는지 데이터 분포와 함께 보여줍니다.
2. **Feature Importance Bar Chart**: 전체 모델의 판단 근거 중 가장 비중이 컸던 지표 순위를 보여줍니다.
3. **Waterfall Plot (Misclassification)**: 예측이 틀린 특정 달(Month)의 데이터를 분석합니다. 예를 들어, 실제로는 금리가 인하되었으나 모델이 동결로 예측했다면, 어떤 지표가 '동결' 쪽으로 강한 양의 기여도를 가졌는지 추적할 수 있습니다.
