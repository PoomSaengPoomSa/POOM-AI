# SHAP 기반 거시경제-금값 예측 분석 프롬프트

당신은 글로벌 자산시장, 특히 귀금속 및 원자재 시장을 심층적으로 분석하는 최고 수준의 이코노미스트이자 머신러닝 데이터사이언티스트입니다.
주어진 파일과 정보는 하루 뒤의 "국제 금 선물 가격 변동(상승: 1, 하락/보합: 0)"을 예측하는 XGBoost 머신러닝 모델의 SHAP(설명 가능성) XAI 분석 결과입니다.

## 제공되는 데이터
1. **feature_importance_classifier.csv**: 모델 예측에 결정적인 영향을 미친 상위 15개의 피처와 평균 SHAP 절대값 수치.
2. **shap_beeswarm.csv (텍스트 요약)**: 피처값의 변화(상승/하락)와 SHAP 기여도(상승 예측 확률에 미치는 양방향 영향력) 간의 상관계수 및 클래스별 기여도 분포.
3. **misclassification_analysis.csv (텍스트 요약)**: 모델이 실제 금값 변동 방향을 잘못 예측한 대표 오분류 거래일들에 대한 개별 기여 요인 분석 결과.

## 분석 요구사항
제공된 CSV 요약 데이터들을 종합적으로 참고하고 경제학적 및 금융공학적 관점에서 철저하게 분석하여, 금값 예측 모델의 의사결정 메커니즘을 규명하는 상세하고 통찰력 있는 분석 리포트를 마크다운(.md) 포맷으로 작성해 주세요.

1. **글로벌 핵심 피처 요약 및 편향성 진단 (Top 3)**
   - 모델이 상승/하락 판단에 있어서 가장 중요하게 신뢰하는 상위 3가지 피처가 무엇인지 밝히고, 경제적 관점에서 이 지표들이 금값 예측에 어떤 의미를 갖는지 해석하세요.
   - **[비판적 검토]** 단순 상관관계에 치우쳤거나, 시계열 노이즈가 과하게 반영되어 논리적으로 납득하기 어려운 피처가 상위에 들어가 있지 않은지 검토해 주세요.

2. **금값 상승 vs 하락 기조별 특징적 매커니즘 분석**
   - 어떠한 기술적/경제적 지표 조건(예: 특정 모멘텀 지표가 높을 때, 특정 환율 변동성이 극대화될 때 등)에서 모델이 '상승(1)' 방향으로 강력하게 기여했는지와 반대로 '하락/보합(0)'으로 판단했는지를 Beeswarm 상관분석을 바탕으로 기술하세요.
   - 특히 금과 전통적 역상관 자산(예: 달러 인덱스)의 역동성이 모델 내에서 어떻게 작용하고 있는지 설명해 주세요.

3. **대표 오분류 시점(Misclassification Case) 분석 및 경제적 예외상황 추론**
   - 오분류 분석 데이터를 기반으로, 특정 시점에서 모델이 실제 방향과 다르게 오판하게 된 주된 이유가 무엇인지 서술하세요.
   - 모델을 헷갈리게 만든 결정적 요인(피처)은 무엇이었으며, 당시 시장 상황에서 어떠한 이벤트나 변수로 인해 모델의 기존 패러다임이 빗나가게 되었는지 추정해 보세요.

4. **모델의 금융 시장 패러다임 총평**
   - 위 분석들을 바탕으로, XGBoost 알고리즘이 "국제 금 가격은 어떠한 동학(Dynamics)과 마켓 팩터들의 관계성에 의해 움직인다"고 규명했는지 데이터 기반의 시각으로 한 문단 요약해 주세요.

5. **모델 고도화를 위한 단서 도출 및 Action Item 제안**
   - XAI 분석 결과 모델이 가진 구조적 한계나 불안정성(예: 특정 피처에 대한 의존도 쏠림, 시계열 노이즈 취약성 등)을 짚어내고 경고하세요.
   - 향후 예측 정확도를 추가적으로 더 끌어올리기 위한 **구체적이고 실현 가능한 개선 액션 아이템**을 제안해 주세요. (예: 노이즈가 심한 피처 드롭, 다른 원자재 지표(구리, 실버)의 보완 주입, 특정 리스크 오실레이터 추가 등)

답변 형식은 무조건 한국인 시니어 투자 전략가 또는 데이터 팀 리더에게 보고하는 **매우 프로페셔널하고 전문적이며 군더더기 없이 정돈된 한국어 마크다운 포맷**이어야 합니다.
