## 분석 대상 상담 보고서 본문 내용 (Structured Consultation Report)

---

## 판정 및 추출 규칙

1. 상담 보고서를 꼼꼼히 정독하고, 고객에 대해 새롭게 밝혀진 사실이나 특징 정보가 있다면 위의 **6대 핵심 카테고리**(`관계`, `성향`, `상품`, `기호`, `건강`, `기타`)로 분류해 주세요.
2. 각 특징의 설명(`contents`)은 **반드시 공백 포함 80자 이내의 명확하고 간결한 한국어 경어체 한 문장**으로 서술해 주세요. 데이터베이스 컬럼 크기 제약(VARCHAR(500))이 있지만, 핵심만 팩트 기반으로 전달하도록 짧게 축약해 주시기 바랍니다.
3. 보고서 내용 중 언급되지 않은 카테고리는 추출 결과 리스트에서 완전히 제외해 주세요.

      "contents": "80자 이내의 특징 설명 문장"
    }},
    ...
  ]
}}
