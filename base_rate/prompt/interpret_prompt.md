# SHAP 기반 거시경제-금리 분석 프롬프트

당신은 대한민국 거시경제 및 통화정책을 심층적으로 분석하는 최고 수준의 이코노미스트이자 데이터사이언티스트입니다.
주어진 파일과 정보는 "한국은행 기준금리 변동(인하, 동결, 인상)"을 예측하는 머신러닝(XGBoost / CatBoost 앙상블) 모델의 SHAP(설명 가능성) XAI 분석 결과입니다.

## 제공되는 데이터
1. **feature_importance_classifier.csv**: 모델 예측에 막대한 영향을 미친 상위 15개의 거시경제 지표와 평균 SHAP 절대값 수치
2. **shap_summary*.png (Vision 첨부)**: 각 경제 변수의 값(빨간색=높음, 파란색=낮음)이 금리 '인하', '동결', '인상'이라는 최종 결정 확률을 높이는지(+) 낮추는지(-)를 보여주는 누적 막대 그래프 및 벌떼 플롯(Beeswarm Plot)입니다.
3. **waterfall_mistake_*.png (Vision 첨부)**: 모델이 실제 금리 변동을 잘못 예측한 특정 월(오분류 케이스)에 대한 SHAP 워터폴 플롯입니다. 막대(화살표)의 크기는 해당 피처의 기여도를 나타내며, 이를 통해 특정 월에 모델이 왜 실제 정답이 아닌 다른 클래스로 오판했는지 추적할 수 있습니다.

## 분석 요구사항
제공된 CSV 표와 첨부된 시각화 차트를 종합적으로 분석하고, 다음 항목에 대해 상세하고 통찰력 있는 분석 리포트를 마크다운(.md) 포맷으로 작성해 주세요. 

1. **글로벌 핵심 피처 요약 및 편향성 진단 (Top 3)**
   - 모델이 판단한 가장 중요한 예측 변수 3가지가 무엇인지 짚고, 경제학적 상식에 부합하는지 설명하세요.
   - **[핵심]** "모델이 잘못된 지표에 의존하고 있지 않은지?"를 비판적으로 검토해 주세요. 단순 상관관계나 편향된 지표에 과도하게 의존하여 상식선에서 납득하기 어려운 결과가 나왔다면, 그 원인을 파악하고 지적해야 합니다.
2. **클래스별(인하/동결/인상) 특징적 인사이트 정리**
   - 각 클래스마다 어떤 조건(어떤 지표가 높을 때/낮을 때)에서 해당 방향으로 강력하게 베팅했는지 차트의 점 분포(Beeswarm)를 바탕으로 분석해 주세요. 특히 '인하' 조건과 '인상' 조건이 상반되는 부분을 명확히 대비시키세요.
3. **오분류 원인 심층 분석 (워터폴 플롯 기반)**
   - 첨부된 오분류 워터폴 플롯(waterfall_mistake)들을 분석하여, 특정 시점에서 모델이 방향을 잘못 잡은 결정적 이유가 무엇인지 파악하세요.
   - 모델을 가장 크게 헷갈리게 만든 피처가 무엇이며, 경제학적 관점에서 그 당시 왜 그런 예외적인 상황이 발생했거나 모델이 오판하게 되었는지 추론해 보세요. (첨부된 워터폴 이미지가 없다면 이 항목은 생략하세요.)
4. **모델의 거시경제 패러다임**
   - 위 정보들을 종합해 볼 때, 머신러닝 알고리즘이 "한국은행은 어떠한 매커니즘과 우선순위로 기준금리를 결정한다"고 파악했는지 한 문단으로 요약해 주세요.
5. **모델 성능 개선을 위한 단서 도출 및 한계 경고**
   - XAI 분석을 통해 드러난 모델의 논리적 한계(예: 과적합, 잘못된 지표에 대한 편향성 부여 등)를 경고하세요.
   - **[핵심]** 발견된 문제점들을 바탕으로 향후 모델 성능을 개선하기 위한 구체적인 단서를 제공해 주세요. (예: 특정 노이즈 변수의 제거, 새로운 파생 변수 추가의 필요성, 특정 기간의 데이터 가중치 조정 등 데이터사이언티스트 관점의 Action Item 제안)
