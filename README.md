# POOM - 금값 변화율 예측 머신러닝 & 퀀트 백테스팅 시스템

우리FISA 6기 AI 엔지니어링반 최종 프로젝트 `POOM` 애플리케이션의 AI 모듈입니다.  
본 시스템은 **오늘의 다양한 거시경제 및 금융 시장 지표들을 학습하여 내일의 금값 변화율(상승/하락 방향성)을 예측하고, 이를 바탕으로 수수료가 적용된 모의 투자 시뮬레이션(백테스팅)을 수행**하는 정교한 퀀트/머신러닝 파이프라인입니다.

---

## 🚀 파이프라인 핵심 성과 요약

* **방향성 예측 정확도 (Hit Rate)**: **58.16%** (Random Forest Classifier 최우수 선정)
  - 기존 수치 예측(회귀) 모델의 노이즈 과적합 한계를 극복하기 위해 **이진 분류(상승 1 / 하락 0) 파이프라인을 병렬 가동**하여 정확도를 **51.52%에서 58.16%로 대폭 개선(+6.64%p)**했습니다.

  - **모델 롱숏 투자 누적 수익률**: **133.24%** 📈
  - **금값 단순 보유 (Buy & Hold) 수익률**: **119.29%**
  - **단순 보유 대비 초과 수익률 (알파)**: **+13.96%p**

├── models/              # [생성 파일] 직렬화된 최우수 모델, 스케일러 및 메타데이터 (.pkl, .json)

## 📊 모듈별 상세 설명

### 1. `data_loader.py` (고차원 피처 엔지니어링)
데이터베이스의 `ml_gold_raw` 테이블을 로드하여 시계열 데이터 누수(Data Leakage)가 없는 **40차원의 풍부한 피처셋**을 빌드합니다.
* **데이터 클렌징**: 거래가 없어서 금값이 `NaN`인 주말 및 공휴일 행을 자동으로 제거하고, 시장 간 휴장일 차이로 발생하는 자잘한 누락치는 앞선 거래일의 값을 채우는 **Forward Fill (`ffill()`)** 처리를 통해 데이터 연속성을 유지합니다.
* **시차 피처 (Lagged Features)**: 금값 및 환율, 유가 등 7개 지표의 **1일 전(t-1), 2일 전(t-2), 3일 전(t-3) 시차 수익률 피처(총 21개)**를 생성하여 시장의 자기상관성과 모멘텀을 학습시킵니다.
* **이동 통계 피처 (Rolling Statistics)**: 단기(**5일**) 및 장기(**20일**) **이동평균(SMA)**과 **이동 표준편차(Rolling Volatility, 변동성)** 피처를 생성하여 추세 및 변동성을 반영합니다.
* **타겟 병렬 추출**: 회귀 타겟(내일의 변화율 실수값)과 분류 타겟(내일 금값 상승 여부 1 or 0)을 동시 생성합니다.

### 2. `eda_visualizer.py` (탐색적 데이터 분석 시각화)
고화질 분석 차트 5종을 `plots/` 폴더에 자동 저장합니다. macOS 환경에 완벽 최적화된 한글 폰트(`AppleGothic`) 설정을 탑재했습니다.
* `eda_correlation_heatmap.png`: 주요 지표 및 기술 지표 간 상관관계 분석 히트맵.
* `eda_gold_trends.png`: 금 절대 가격 추세선과 변화율 변동성 비교 차트.
* `eda_target_distribution.png`: 예측 타겟 변화율의 통계 요약 및 정규분포 대조 곡선.
* `eda_class_balance.png`: 상승(1) vs 하락/보합(0) 분류 클래스 균형 차트.
* `eda_scatter_plots.png`: 금값, DXY, S&P500, VIX 변화율과 타겟 간의 산점도 및 선형 회귀 추세선.

### 3. `baseline_model.py` (병렬 모델 학습 및 선정)
* **시계열 순차 분할**: 미래 정보 누수 차단을 위해 랜덤 셔플링 대신 연대기 순으로 앞의 **80% (2,473개 거래일)**를 훈련 세트, 뒷부분 **20% (619개 거래일)**를 테스트 세트로 철저히 양분했습니다.
* **6종 알고리즘 병렬 훈련**: 
  - *회귀 모델 3종*: Ridge Regression, Random Forest Regressor, Gradient Boosting Regressor
  - *분류 모델 3종*: Logistic Regression, Random Forest Classifier, Gradient Boosting Classifier
* **최적 모델 선정 및 저장**: 테스트 셋 방향성 정확도(Accuracy / Hit Rate)가 가장 뛰어난 **Random Forest Classifier**를 최종 🏆 최우수 모델로 선정하고, 최적화 파일(`best_baseline_model.pkl`), 스케일러(`scaler.pkl`), 모델 정보 메타데이터(`model_meta.json`)를 `models/` 폴더에 일괄 직렬화 저장합니다.

### 4. `evaluator.py` (유연한 평가 및 퀀트 백테스팅 엔진)
* **맞춤형 메타 판독**: `model_meta.json`을 읽고 모델 유형에 맞추어 평가지표를 자동 산출합니다.
* **백테스팅 시뮬레이션**: 모델의 익일 상승/하락 예측 결과에 맞춰 롱숏(+1/-1) 포지션을 스위칭하고, **0.05%의 현실적인 편도 거래 수수료를 차감한 실질 복리 수익률**을 연산합니다.
* **`plots/evaluation_results.png` 출력**:
  - 모델 롱숏 전략 vs 금값 단순 보유의 **누적 수익률 시계열 비교 차트**.
  - 오차 행렬 열지도(Confusion Matrix Heatmap) 및 ROC 곡선 차트.

### 5. `run_pipeline.py` (통합 실행 마스터)
* 전체 3단계 파이프라인(EDA ➡️ 모델 비교 훈련 및 최우수 선발 ➡️ 최종 퀀트 백테스트 검증)을 차례대로 수행해 주는 일체형 구동기입니다.
